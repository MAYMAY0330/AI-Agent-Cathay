# 確認Smart Event 簽到系統與國泰人壽APP串接適法性問題



法務室意見

緣　貴單位(客群經營部專案企劃科)為提高公司活動簽到效率及客戶體驗，擬規劃建置SMART EVENT數位簽到系統，並與國泰人壽APP串接完成簽到，爰詢問:(一)客戶登入國泰人壽APP，點擊保戶卡產出身分識別QR CODE，SMART EVENT系統掃描後，自動將遮蔽後之客戶姓名及電話寫入報名表，以完成簽到，是否可行？(二)參與活動之公關公司以統一編號及密碼登入SMART EVENT系統後，可查詢該場次經遮蔽後之客戶個資，上開作法是否有適法性之疑慮？本室謹復意見如下。

問題(一)部分：

按「本法用詞，定義如下：五、利用：指將蒐集之個人資料為處理以外之使用。」、「非公務機關對個人資料之利用，除第六條第一項所規定資料外，應於蒐集之特定目的必要範圍內為之。但有下列情形之一者，得為特定目的外之利用：六、經當事人同意。」，個人資料保護法(下稱個資法)第2條第5款及第20條第1項第6款定有明文。

次按「國泰人壽及其關係企業可能會在遵守本隱私權政策的前提下，相互共享本項個人資訊，也可能結合其他資訊，用於提供及優化我們的商品、服務及或活動。」、「我們可能會使用包括但不限於您的生日在內等個人資訊來驗證身分、協助辨識使用者及確認適當的服務。」、「若你同意參加抽獎、競賽或類似促銷活動，我們也可能使用您提供的資訊進行這些活動。」，國泰人壽APP會員註冊隱私權條款約定有明文(參附件)。

查客戶點擊保戶卡產出身分識別QR CODE，公司以SMART EVENT系統掃描核對客戶個資後，自動將客戶姓名及電話寫入報名表簽到，核屬對個人資料之利用行為，依前揭個資法之規定，需經客戶同意始得為之。

據前開隱私權條款之約定，客戶於註冊國泰人壽APP時，應得認已同意公司於提供客戶服務或活動之特定範圍內得使用其個人資料，故公司作法應無違反個資法之虞。

問題(二)部分：

按「國泰人壽可能會將特定的個人資訊，提供予國泰人壽合作之服務、行銷供應商等合作夥伴，其使用目的僅限於提供或優化我們的商品、服務…」，國泰人壽APP會員註冊隱私權條款約定有明文。

查公關公司於業務範圍內為公司之使用人，公司於公務上需要將客戶個人資料提供該公司查閱，核屬於特定目的範圍內之內部使用，且依上開隱私權條款約定，客戶於註冊國泰人壽APP時，應得認已同意公司將其個人資料提供合作商使用，故應無適法性疑義。
