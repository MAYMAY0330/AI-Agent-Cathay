# 法務室意見

一、緣公司為配合數位時代轉型及保障型商品推動,擬改版現行 Cathay Box,除既有業務管理功能外,新增「客戶互動行銷」視角,貴單位(數據經營部客戶智能科)爰詢問:若客戶有開啟報告書,系統將於業務管理視角提供客戶瀏覽資訊予業務同仁(瀏覽項目如下:開啟報告書、點擊保障項目、點擊商品組合、點擊行銷素材及呼叫顧問服務),有無違反個人資料保護法(下稱個資法)之虞?本室謹復意見如下。

二、按「本法用詞,定義如下:一、個人資料:指自然人之姓名、…社會活動及其他得以直接或間接方式識別該個人之資料。」「個人資料之蒐集、處理或利用,應尊重當事人之權益,依誠實及信用方法為之,不得逾越特定目的之必要範圍,並應與蒐集之目的具有正當合理之關聯。」「非公務機關對個人資料之利用,除第六條第一項所規定資料外,應於蒐集之特定目的必要範圍內為之。」個資法第2條第1款、第5條及第20條第1項本文定有明文。

三、茲此,「網頁式報告書」中蒐整之客戶資訊,倘已取得當事人同意公司為蒐集、處理及利用,且其利用與蒐集之目的具有正當合理之關聯者,則於法尚無不合,先予敘明。

四、再者,依  貴單位檢附之BOX改版功能說明中「業務管理視角」之系統畫面,客戶「開啟報告書」、「點擊保障項目」、「點擊商品組合」及「點擊行銷素材」之動態資訊,似屬報告書暨其內容之簽收、閱啟等紀錄,另「呼叫顧問服務」係客戶主動請求提供服務之紀錄,與當事人之個人資料尚屬有間,應尚無適法性疑義。✓ 110.8
