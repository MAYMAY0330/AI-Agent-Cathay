# 法務室意見

一、緣 貴單位(客群經營部專案企劃科)為避免因服務人員異動而產生服務斷點,擬將前任服務人員建立於 Cathay Box APP 之既有客戶拜訪紀錄(如拜訪日期、拜訪項目、活動日誌等,以下合稱拜訪資料),開放予現任服務人員及業務主管查詢,爰詢問上開做法有無違法性疑義?本室謹覆意見如下。

二、按「本法用詞,定義如下:一、個人資料:指自然人之姓名、出生年月日、國民身分證統一編號、…、婚姻、家庭、教育、職業、…聯絡方式、財務情況、社會活動及其他得以直接或間接方式識別該個人之資料。…五、利用:指蒐集之個人資料為處理以外之使用。」「非公務機關對個人資料之利用,除第六條第一項所規定資料外,應於蒐集之特定目的必要範圍內為之。」,個人資料保護法(下稱個資法)第 2 條第 1 款、第 5 款及第 20 條第 1 項本文分別定有明文。

三、查前任服務人員輸入之客戶基本資料、財務狀況及社會活動等紀錄,核屬客戶個人資料,惟公司如基於客戶服務需要而授權現任服務人員及業務主管查詢,係屬公司內部之利用,且與蒐集之特定目的相符,應無違法性疑慮,餘請參本室 105.7.25 法律意見(如附件)。

四、另有關前任服務人員輸入之預計拜訪日期、拜訪項目及部分拜訪內容,僅係該員拜訪客戶前預定日標之紀錄,非屬客戶之個人資料,自無個資法之適用,併此敘明。

法務室意見

一、按個人資料保護法第2條第5款、第20條1項分別規定:「本法用詞,定義如下:......五、利用:指將蒐集之個人資料為處理以外之使用。......」、「非公務機關對個人資料之利用,除第六條第一項所規定資料外,應於蒐集之特定目的必要範圍內為之」。

二、準此,本公司駐點業務人員查詢圓險客戶之資料,核屬本公司對於個人資料之利用,故應依誠實及信用方法為之,且應於蒐集之特定目的必要範圍。

三、另因金融控股公司法(第42條及第60條第12款參照)及個人資料保護法(第27條、第29條、第41條、第47條、第48條參照)均有相當重的罰則,建議 貴單位如要開放駐點業務人員查詢圓險件客戶之圓險資料,應加強相關人員之保密教育訓練,確實遵守公司相關規定,免滋爭議。
