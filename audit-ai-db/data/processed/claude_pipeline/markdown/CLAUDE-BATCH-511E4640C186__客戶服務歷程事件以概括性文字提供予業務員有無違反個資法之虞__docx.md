# 客戶服務歷程事件以概括性文字提供予業務員有無違反個資法之虞



法務室意見

緣公司為提升業務員工作效率及因應數位轉型計畫，已啟動業務員Cathay Box改版及CRM2.0專案，現擬運用公司建置之「客戶服務歷程資料庫」，將客戶之適宜服務歷程事件，以概括性文字描述之方式，提供予具經手服務關係之業務同仁。貴單位(數據經營部客戶智能科)爰詢問：前揭規劃有無違反個人資料保護法(下稱個資法)之虞？本室謹復意見如下。

按「本法用詞，定義如下：一、個人資料：指自然人之姓名、…聯絡方式、財務情況、社會活動及其他得以直接或間接方式識別該個人之資料。」、「個人資料之蒐集、處理或利用，應尊重當事人之權益，依誠實及信用方法為之，不得逾越特定目的之必要範圍，並應與蒐集之目的具有正當合理之關聯。」、「非公務機關對個人資料之利用，除第六條第一項所規定資料外，應於蒐集之特定目的必要範圍內為之。」個資法第2條第1款、第5條及第20條第1項本文定有明文。

查貴單位檢附之附件中有關提供業務同仁之文字訊息欄位，含客戶姓名及保單號碼，核屬客戶個人資料，倘已取得當事人同意公司為蒐集、處理及利用，且其利用與蒐集之目的具有正當合理之關聯者，則於法尚無不合。復公司如基於客戶服務需要而提供予經手服務關係之業務同仁知悉，係屬公司內部之利用，應尚無適法性疑義，惟須注意應依誠實及信用方法為之。
