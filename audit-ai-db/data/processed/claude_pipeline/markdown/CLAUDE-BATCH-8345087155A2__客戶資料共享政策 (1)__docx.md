# 客戶資料共享政策 (1)



國泰人壽保險公司客戶資料共享政策

113年3月5日修正

權責單位:數據暨人工智慧發展部

第一條 (目的及依據)

國泰人壽保險股份有限公司（以下簡稱「本公司」）及國泰金融控股公司(以下簡稱「國泰金控」及其金融機構子公司(以下與本公司合稱為「集團各公司」)為發展集團業務、提升客戶服務、控管風險、促進本公司或集團各公司與其他金融機構間業務合作等目的，辦理客戶資料共享，爰依「金融機構間資料共享指引」(以下簡稱「共享指引」) 及「國泰金控暨子公司客戶資料共享政策」等相關法令規定，訂定「國泰人壽保險公司客戶資料共享政策」 (以下簡稱「本政策」) ，以資遵循。

第二條 (管理單位及授權)

本政策之權責管理單位為數據暨人工智慧發展部，除法令或內部規範另有規定外，客戶資料共享相關事項，由數據暨人工智慧發展部統籌規劃管理，並授權該單位得依權責及管理需要，訂定執行作業規範。

第三條 (適用範圍及定義)

本公司與集團各公司或非本集團之金融機構間，依其他法令共享客戶資料者，應從其規定辦理，並應遵循本政策之各項規範；非依其他法令共享客戶資料者，應依共享指引及個人資料保護法(以下簡稱「個資法」)之規定辦理，並應遵循本政策之各項規範。

本政策各項名詞定義如下，後續如共享指引及主管機關相關規定修正時，從其最新規定辦理：

集團各公司：本政策所稱集團各公司不包括集團各公司之國外從屬公司或分公司，但法令另有規定者不在此限。

金融機構：係指依本國法設立之金融控股公司、銀行、信用合作社、票券金融公司、信託業、電子支付機構、辦理郵政儲金匯兌業務或簡易人壽保險業務之郵政機構、證券商、證券投資信託事業、證券金融事業、證券投資顧問事業、期貨商、槓桿交易商、期貨信託事業、期貨經理事業、期貨顧問事業及保險公司，或外國金融機構在臺分公司。金融機構不包括前述公司之國外從屬公司或分公司，但法令另有規定者不在此限。

其他法令：得參考主管機關揭露於網站之法令，未來其他法令如有調整或新增，主管機關未即時於網站更新者，則自該法令生效之日起適用。

第四條 (客戶資料共享範圍)

依共享指引及個資法共享客戶資料者，得共享之客戶資料範圍，包括客戶基本資料、身分核驗資料、帳戶資料、金融商品或服務之交易紀錄、負面資訊、認識客戶(KYC)資料、金融機構加值之資料、電子通訊歷程紀錄(如IP位址)、其他經客戶及合作金融機構依法同意共享之客戶資料。

依其他法令共享客戶資料者，得共享之客戶資料範圍，依所依據之其他法令規定。

第一項所稱「加值資料」係指金融機構基於客戶提供之資料所直接或間接創建之推論資料及衍生資料。

第五條 (內部控制規範)

本公司依本政策資料共享時，應依相關法令及以風險為基礎，建立下列之內部控制：

辦理資料共享，應有明確、妥適目的，並應符合相關法令規定及注意倫理道德。

共享之資料如屬身分核驗資料或負面資訊等，對客戶權益有較大影響性者，應審慎辦理並為相關必要措施。

資料共享過程中，應確保參與者及其員工均取得合法授權。

資料共享之日常維護、留存、權限設定、報表管理、共享結束後之處理、內部審查及分層負責等事宜，應有相關管理措施。

資料共享所涉之資訊系統及資料傳輸，應具備安全性，確保個人資料與客戶權益。

資料共享如涉及對外客戶服務作業時，應訂定受理客戶申訴及處理爭議之內部標準程序。

前項各款之內控措施，應由本公司涉及資料共享之各業務單位依本公司內控制度之規定，會辦相關內控權責單位妥善安排，並於必要時，訂定執行細節、後續修正及內部規範等，悉依本公司內控制度及分層負責表，辦理之。

第六條 (資料保護)

本公司辦理資料共享時，相關客戶資料應有適當管控機制，以確保其保密性，並施以適當之管理措施，確保相關人員對資料共享之相關法令、內規、共享約定條件等有充足之認識。

第七條 (附則)

本政策如有未盡事宜，悉依相關法令及本公司相關內部規範辦理之。

本政策經董事會核定後施行，修正或廢止時亦同。

本政策訂於111年5月13日，歷次修正紀錄如下:113年3月5日。
