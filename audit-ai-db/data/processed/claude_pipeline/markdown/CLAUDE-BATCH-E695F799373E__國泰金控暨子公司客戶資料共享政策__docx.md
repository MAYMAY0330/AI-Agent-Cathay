# 國泰金控暨子公司客戶資料共享政策



國泰金控暨子公司客戶資料共享政策

111.3.11訂定

權責單位: 數位數據暨科技發展中心數據科技發展部

第一條 (目的及依據)

國泰金融控股股份有限公司（以下簡稱「本公司」）及其金融機構子公司(以下與本公司合稱為「集團各公司」)為發展集團業務、提升客戶服務、控管風險、促進本集團與其他金融機構間業務合作等目的，辦理客戶資料共享，爰依「金融機構間資料共享指引」(以下簡稱「共享指引」) 及相關法令規定，訂定「國泰金控暨子公司客戶資料共享政策」 (以下簡稱「本政策」) ，以資集團各公司共同遵循。

第二條 (管理單位及授權)

本政策之權責管理單位為數位數據暨科技發展中心數據科技發展部(以下簡稱「金控數據部」)，除法令或內部規範另有規定外，客戶資料共享相關事項，由金控數據部統籌規劃管理，並授權該單位得依權責及管理需要，訂定執行作業規範。

第三條 (適用範圍及定義)

集團各公司間，或集團各公司與非本集團之金融機構間，依其他法令共享客戶資料者，應從其規定辦理，並應遵循本政策之各項規範；非依其他法令共享客戶資料者，應依共享指引及個人資料保護法(以下簡稱「個資法」)之規定辦理，並應遵循本政策之各項規範。

本政策各項名詞定義如下，後續如「共享指引」及主管機關相關規定修正時，從其最新規定辦理：

集團各公司：本政策所稱集團各公司不包括集團各公司之國外從屬公司或分公司，但法令另有規定者不在此限。

金融機構：係指依本國法設立之金融控股公司、銀行、信用合作社、票券金融公司、信託業、電子支付機構、辦理郵政儲金匯兌業務或簡易人壽保險業務之郵政機構、證券商、證券投資信託事業、證券金融事業、證券投資顧問事業、期貨商、槓桿交易商、期貨信託事業、期貨經理事業、期貨顧問事業及保險公司，或外國金融機構在台分公司。金融機構不包括前述公司之國外從屬公司或分公司，但法令另有規定者不在此限。

其他法令：得參考主管機關揭露於網站之法令，未來其他法令如有調整或新增，主管機關未即時於網站更新者，則自該法令生效之日起適用。

第四條 (客戶資料共享範圍)

依共享指引及個資法共享客戶資料者，得共享之客戶資料範圍，包括客戶基本資料、身分核驗資料、帳戶資料、金融商品或服務之交易紀錄、負面資訊、KYC資料、金融機構加值之資料、電子通訊歷程紀錄(如IP 位址)、經客戶及合作金融機構依法同意共享之客戶資料。

依其他法令共享客戶資料者，得共享之客戶資料範圍，依所依據之其他法令規定。

第五條 (內控制度)

集團各公司依本政策資料共享時，應依相關法令及以風險為基礎，建立下列之內部控制：

辦理資料共享，應有明確、妥適目的，並應符合相關法令規定及注意倫理道德。

共享之資料如屬身分核驗資料或負面資訊等，對客戶權益有較大影響性者，各公司應審慎辦理並為相關必要措施。

資料共享過程中，應確保參與者及其員工均取得合法授權。

資料共享之日常維護、留存、權限設定、報表管理、共享結束後之處理、內部審查及分層負責等事宜，應有相關管理措施。

資料共享所涉之資訊系統及資料傳輸，應具備安全性，確保個人資料與客戶權益。

資料共享如涉及對外客戶服務作業時，應訂定受理客戶申訴及處理爭議之內部標準程序。

前項各款之內控措施，應由各公司資料共享主辦單位依該公司內控制度之規定，會辦相關內控權責單位妥善安排，並於必要時，訂定執行細節、後續修正及內部規範等，悉依該公司內控制度及分層負責表，辦理之。

第六條 (資料保護)

各公司辦理資料共享時，相關客戶資料應有適當管控機制，以確保其保密性，並施以適當之管理措施，確保相關人員對資料共享之相關法令、內規、共享約定條件等有充足之認識。

第七條 (子公司管理)

子公司如有辦理客戶資料共享，應參照本政策及其相關法令，訂定該公司客戶資料共享政策，提報該公司董事會通過，妥善辦理。

嗣後本政策如有修正者，亦同。

第八條 (附則)

本政策如有未盡事宜，悉依相關法令及公司內部規範辦理之。

本政策之訂定、修正或廢止應經董事會同意。

本政策自訂定日起生效施行，修正或廢止時亦同。
