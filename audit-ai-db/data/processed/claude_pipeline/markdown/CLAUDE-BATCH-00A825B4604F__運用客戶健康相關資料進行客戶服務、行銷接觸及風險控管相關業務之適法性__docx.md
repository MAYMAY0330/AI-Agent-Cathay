# 運用客戶健康相關資料進行客戶服務、行銷接觸及風險控管相關業務之適法性



法務室意見

緣本公司前曾舉辦多個專案蒐集客戶健康相關資料，  貴單位(即客群經營部資料探勘科)茲詢問：(一)是否可運用各專案已蒐集之客戶健康相關資料，供各部室進行跨活動、跨業務別之利用(例如進行客戶服務、行銷接觸及風險控管等)？(二)蒐資之健康活動若為網路/APP活動，是否可於活動報名流程中，增設「同意本公司利用其特種個資」之選項供當事人點選，並加註「被保險人參加本次活動，同意以電子文件之方式取代書面同意」？本室謹覆意見如下。

關於問題(一)部分：

按個人資料保護法(下稱個資法)第6條第1項第6款、第20條第1項第6款規定：「有關病歷、醫療、基因、性生活、健康檢查及犯罪前科之個人資料，不得蒐集、處理或利用。但有下列情形之一者，不在此限：六、經當事人書面同意。但逾越特定目的之必要範圍或其他法律另有限制不得僅依當事人書面同意蒐集、處理或利用，或其同意違反其意願者，不在此限。」、「非公務機關對個人資料之利用，除第六條第一項所規定資料外，應於蒐集之特定目的必要範圍內為之。但有下列情形之一者，得為特定目的外之利用：六、經當事人同意。」

爰此，公司基於各專案所蒐集之個人資料，如利用於該專案以外，恐已涉及特定目的外利用，依前揭個資法規定，須經當事人同意。至於各專案於蒐集個資時，是否已得當事人同意公司為特定目的外利用，須逐一檢視該專案本身之特定目的範圍及個資約定條款而定，尚難一概而論。

末依本公司「個人資料蒐集、處理與利用管理辦法」第11條第2項規定，如欲利用收集之個人資料而為特定目的外之利用時，應向風險管理二部作業風險管理科申請，由該單位審核，提請  貴單位注意。

關於問題(二)部分：

按個人資料保護法施行細則第14條規定：「本法第六條第一項但書第六款、第十一條第二項及第三項但書所定當事人書面同意之方式，依電子簽章法之規定，得以電子文件為之。」

次按保險法第一百七十七條之一第二項所定本人書面同意方式、業務範圍及其他應遵行事項管理辦法第2條第1項規定：「本法第一百七十七條之一第一項所稱書面同意方式，應以紙本或符合電子簽章法之電子文件方式為之。」

爰此，蒐集、處理及利用特種個資之書面同意，得以電子文件為之，餘請參考先前本室及律師意見(如附件)。另建議於有關個資約定之條款中敘明欲收集之特種個人資料項目，例如係病歷或健檢資料等，以免日後衍生爭議。
