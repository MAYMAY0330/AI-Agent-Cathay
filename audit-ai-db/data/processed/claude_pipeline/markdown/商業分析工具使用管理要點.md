# 商業分析工具使用管理要點



國泰人壽保險公司商業分析工具使用管理要點

113年4月19日修訂

權責單位：數據暨人工智慧發展部

目的

為規範本公司商業分析工具之使用，進行商業分析或資料視覺化時，確保商業分析平台(下稱本平台)安全性及權責歸屬，有效管理相關使用及維運作業，特訂定本要點。

適用對象

本公司有商業分析需求，並經申請核准使用本平台相關工具進行開發者，應依本要點之規定辦理。本要點未規定者，依其他有關規章之規定辦理。

商業分析平台

所稱「商業分析平台」係指本公司提供商業分析統計及資料視覺化的工具或環境(包含但不限於SAS-EG、IBM Cognos、Tableau系統等)，

本平台環境，包括運行伺服器、儲存空間等，經申請後使用本平台相關之資源，及本平台另外產生的複合式環境均屬之。

用詞定義及單位權責

本要點相關用詞定義及單位權責說明如下：

一、平台規劃單位：指相關平台規劃(包含申請流程、平台使用方法等)之規劃管理單位，由數據暨人工智慧發展部負責。

二、平台維護單位：指本平台之建置及管理維運單位，由行銷資訊部資料倉儲科負責。

三、平台使用單位：指因業務目的，有使用本平台需求之單位。

使用者帳號申請

使用本平台應先透過表單管理系統申請本平台相關分析工具帳號，經科主管以上主管核准後，始得依業務所需，使用本平台分析環境。

平台使用限制

使用依第五條規定申請核准之帳號，始得登入本平台進行分析。如為安裝於個人電腦之第三方軟體工具，由客戶端進行驗證登入，例如 SAS-EG、Tableau Desktop；如為伺服器版工具，則應透過單一入口登入(SSO) 連線使用，例如 Cognos、Tableau Server。

本平台得以透過連線方式取得資料庫數據，連線帳號之資料權限，依照使用者進行限制，資料表權責相關規章依「國泰人壽數據平台使用管理要點」之規定辦理。

安全管理

平台維護單位應於不同分析工具環境，分別保存分析工具使用紀錄，並定期查驗紀錄檔是否有效。

平台使用單位可於分析工具上申請分析資料暫存目錄，原則上依科級單位建立目錄，並授權同單位同仁存取，如有單位異動者應同步移除其權限。

如因跨部門合作專案需求，得由專案負責人檢附專案成立目的、專案組成單位等規劃文件，提出電腦作業申請書，向平台維護單位申請建置專案目錄提供跨部門共用。專案負責人並應於專案結束後另提電腦作業申請書停止專案目錄之使用。

平台使用單位就暫存目錄所取得之資料，應善盡保管義務並定期清查，針對已屆處理或利用期限之前述資料，應確實辦理銷毀作業，以確保資料安全，及使用空間之有效利用  。

平台使用單位透過分析工具取得或產生的複合式資訊，應遵循「國泰人壽保險公司資料處理作業安全管理規則」之規定，確保公司資料或客戶個資檔案之安全性。

保密義務

本平台之維護人員、開發人員、使用人員、設備管理人員(以下合稱平台相關人員)，於接觸或使用公司資料時，不得有洩漏或移作他用之情事，並須依相關規定使用本平台，平台相關人員所屬單位之部室主管負有控管該平台之責任，並應嚴守保密義務。

附則

本要點如有未盡事宜，悉依相關法令、國泰金融控股股份有限公司資料倉儲管理要點、國泰金融控股股份有限公司資訊作業申請管理要點、國泰金融控股股份有限公司檔案傳輸要點及本公司相關規定辦理。

本公司員工違反本要點致影響公司或保戶權益，經查證屬實者，除移送人事管理單位議處外，行為人並應自負相關民、刑事責任。

施行日期

本要點呈總經理核准後施行，修正或廢止時亦同。
本要點訂於109年11月13日，歷次修正紀錄如下：112年12月18日、113年1月30日、113年4月19日。
