# 附件一、NPS電話拜訪記錄cookie政策修正

## 歐盟即將實施的數據隱私法規文件

### 電子隱私法規（ePrivacy Regulation）

• 管轄對象：電子通訊服務提供者、網站營運商、電郵行銷廠商和cookie使用者
• 除資料控制者和處理者，資料主體權利延伸至電子通訊終端設備之使用者（device用戶）
• 網站若要使用cookie，需事先取得用戶明示同意（explicit consent）
• 須明確告知用戶cookie的使用目的、期限以及如何移除cookie等資訊

## 主要異動

### 說明

目前網站已標註使用cookie技術，依據我國個資法規定，該等cookie蒐集之資料屬個資範疇。為確保用戶知情權，網站應制訂cookie政策並告知用戶

### 電子隱私法規（ePrivacy Regulation）重點

• 任何網站若要蒐集或使用cookie，必須事先取得用戶明示同意
• 須明確說明使用目的、蒐集哪些資料及資料保存期限
• 須提供用戶選擇是否允許使用cookie的選項

### 建議處理方式

企劃部

• 規劃cookie蒐集使用同意書文字
• 增加cookie使用說明頁面

行銷部

• 宣導cookie政策上線訊息

資訊部

• 調整網站前端顯示
• 增加cookie政策頁面

法務室意見

• 電子隱私法規目前仍在歐盟審議中，尚未生效
• 建議可先參考現行歐盟cookie指令規範，先行規劃相關機制

結論

依法務室建議，請企劃部先行規劃cookie使用說明及同意機制，資訊部配合調整網站顯示方式
