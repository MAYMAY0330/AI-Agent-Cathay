# 金控同仁兼任國壽職務於國壽辦公室進行資料蒐集分析疑義

## Page 1

# 法務室意見

一、緣  貴單位(客群經營部專案企劃科)因國套金控(下稱金控)及本公司為兼務需求,擬由金控掛名兼任客群經營部職務,兼任期間將使用本公司資料庫進行客戶個人資料之處理及利用,爰詢問該兼任同仁得否於金控辦公室辦公,或應於本公司辦公室辦公?本室謹覆意見如下。

二、按「金融控股公司或其子公司人員,除法令另有規定外,得兼為相關子公司之業務執行或商品銷售行為,並應先取得法定之特定資格或證照。」,金融控股公司及其子公司自律規範第17條第1項定有明文。

三、茲此,因本件所詢兼任人員之原任及兼任職務均為一般職員而非經理人,即與前揭規定無違,應無違法性問題(詳參本室99.4.2法律意見,附件1),合先敘明。

四、查金控人員兼任本公司職務,其於執行本公司職務時,即為本公司之使用人,本公司授權其使用客戶資料庫,核屬公司內部對個人資料之處理及利用,應無違法性疑慮,餘請參本室106.2.17法律意見(附件2)。

五、另經詢  貴單位表示,本公司之客戶資料庫設有防火牆,僅本公司員工且有獲得授權者方可使用。茲此,於本公司有兼任職務之金控人員,如以本公司員工身分並於授權範圍內使用資料庫,應不得合新揭個資法規「依有個人資料應採行適當安全措施」之規定。其如須於金控辦公室辦公,應尚無不可,惟本公司仍應注意個人資料之保密,並負管控之責,不得有洩漏或由非本公司業務相關人員取得之情。✓ 10.26 / 10.17

# 法務室意見

一、按金融控股公司法第4條第1項第4款第4目規定:「四、子公司
   指下列公司:…(四)金融控股公司持有已發行有表決權股份總
   數或資本總額超過百分之五十,或其過半數之董事由金融控股公
   司直接、間接選任或指派之其他公司。」。

二、次按「…負責人之範圍,包括董事、監察人、總經理、副總、經
   理、協理、經理或與其職責相當之人。」、「有下列情事之一,不
   得充任保險業之負責人:十三、擔任其他保險業、金融控股公司、
   銀行、信託公司…。但下列情形,不在此限:(二)保險業為金融
   控股公司子公司者,其負責人得兼任該控股公司或其他子公司之
   負責人。但子公司間不得有經理人互相兼任之情事。」、保險業與
   利害關係人從事放款以外之其他交易管理辦法第2條第2項、保
   險業負責人應具備資格條件準則第3條第1項第13款第2目分別
   定有明文。

三、查百慕達國泰保險公司(下稱百慕達國泰)之董事皆為國泰人壽所
   指派;又國泰人壽為國泰金控之保險子公司,是國泰金控得間接
   選任或指派百慕達國泰過半數之董事,從而,百慕達國泰亦應為
   國泰金控之保險子公司。

四、復查百慕達國泰之公司章則明列各部門之職掌事項、作業準則及
   分層負責表等,是各部之部長就章則所列職掌事項應有為公司管
   理事務及簽名之權,故渠等為經理人(民法第553條第1項、公司
   法第31條第2項參照),應無疑義。從而,百慕達國泰之行政暨
   投資部部長兼任國泰人壽企劃部經理,恐與前揭保險業負責人應
   具備資格條件準則之規定未合。

# 法務室意見

一、據貴單位表示:現行金控人事系統為金控、人壽、投顧及創投等四家公司共同使用,並由人壽IT代為維運,現為因應集團需求擴大,擬新增部分子公司及孫公司共同使用該系統,並委由人壽IT負責開發,開發完成後委由人壽IT負責維運,茲詢問:集團人事系統共用設計及維運方式是否有資安或法令上之疑慮?

二、按個人資料保護法(下稱個資法)第27條規定:「非公務機關保有個人資料檔案者,應採行適當之安全措施,防止個人資料被竊取、竄改、毀損、滅失或洩漏。」。

三、依貴單位所述,本公司所有之個人資料檔案雖置於金控各子公司共用之資料庫,惟資料庫已設定權限避免各子公司使用者存取其他公司之個人資料。依前揭個資法規定,倘資料庫之權限控管措施屬安全、適當,則應無違反個資法之疑慮。

四、次按個資法第4條規定「受公務機關或非公務機關委託蒐集、處理或利用個人資料者,於本法適用範圍內,視同委託機關。」。查負責維運管理共用系統之人員,核屬各子公司之使用人,其因系統維運管理所必要而接觸各子公司之個人資料,應無違法性問題,惟提請注意有關金控集團子公司間之資訊設備共用,應依「國泰金融控股股份有限暨各子公司間資訊共同作業管理辦法」(如附件一)相關規定辦理(例如:視需要訂定相關合約及遵守保密義務)。

五、未按金融控股公司旗下子公司進行 IT 資源整合併進行費用分攤,涉及利害關係人交易(如附件二,行政院金融監督管理委員會98年3月6日金管銀(一)字第09700507710號函),提請注意依該相關規定辦理。
