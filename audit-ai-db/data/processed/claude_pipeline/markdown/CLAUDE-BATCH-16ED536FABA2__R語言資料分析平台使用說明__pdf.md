二、 資料表格權限申請 
欲申請R 語言資料分析平台讀取資料庫表格權限，請透過公司現行「電

2. 新增R 語言資料分析平台表格 
若欲讀取自主分析、經營情報資料庫以外的資料表，請提電腦作業申
請書同時給上游資料單位與資料倉儲科，以利上游提供資料由倉儲科
匯入R 語言資料分析平台，並請提供該資料表相關資訊，申請書格式
請參考下方說明，導入之資料格式預設為文字檔(如TXT 或CSV 檔)，
若須轉為XDF 格式，請另外說明轉置目的。 
 
XDF 格式說明： 
XDF 為一使用Block 方式儲存欄與列之資料格式，對於大量資料處理
能以較佳效率化方式執行，但R 程式需搭配使用微軟Revolution R 相
關函數才能發揮效益，詳見微軟官方文件說明，目前已轉XDF 格式資
料表格請見http://cxlsvr248/ad_hoc/main.asp (請選擇R 語言資料分析
平台頁面)。

四、 資料及管理單位一覽表 
欲查詢現行R 語言資料分析平台資料或資料權責單位，請至：國泰員工
入口網站\系統專區\資料分析\自主分析資料庫系統-資料表查詢，並選擇
「R 語言資料分析平台」進入。 
 
 
 
五、 注意事項 
1. 人員單位異動後，R 語言資料分析平台之使用權限須按本規範重新進
行申請。 
2. 基於資安考量，每半年由商業智能科以照會單向各科清查R 語言資料
分析平台權限及相關資料表使用權限需求。
