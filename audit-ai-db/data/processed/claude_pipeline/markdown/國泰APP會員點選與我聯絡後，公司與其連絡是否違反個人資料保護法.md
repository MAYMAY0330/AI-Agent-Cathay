# 法務室意見

一、 貴單位所詢問題為:國泰優惠App(下稱App)之會員,若點擊App內廣告商品「同意專人與我聯絡」之按鈕,系統會透過國泰金控之委外廠商(紀錄會員在 APP 點擊之內容及次數係由委外廠商負責)留存該會員之邀請碼(非個資之8碼序號),委外廠商將蒐集之邀請碼傳輸予公司IT部門,由本公司IT部門比對該邀請碼後,將對應之會員姓名與電話提供予直效行銷部行銷保險商品,是否有違個人資料保護法(下稱個資法)之疑義,本室意見如下。

二、 按個人資料保護法第20條規定:「非公務機關對個人資料之利用,除第六條第一項所規定資料外,應於蒐集之特定目的必要範圍內為之。」。

三、 查國泰優惠App會員權益(含個資保護宣告)約定條款(下稱約定條款)第11條約定:「甲方¹在提供會員相關服務、專屬優惠、商品行銷,......及其他法令許可的目的範圍內(下稱蒐集目的),得將會員於『國泰優惠App』及本服務中留存之個人資料(包括但不限於姓名、身分證字號、出生年/月/日、email、連絡電話、所屬甲方各公司之身分註記、基於提供本服務所需資料等)......提供給甲方各公司......蒐集、處理、利用及國際傳輸。」。

四、 依前揭約定條款約定,會員於註冊APP時,已同意本公司得於商品行銷之特定目的內,利用其個人資料,爰此,倘APP會員了解特定廣告商品並同意指派專人與其連絡時,本公司將客戶之姓名與電話交由直效行銷部撥打電話等行為,核屬就個人資料蒐集目的範圍內之利用,與個資法規定尚無違背。
