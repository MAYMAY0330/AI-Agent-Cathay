# 個人資訊的蒐集與使用

個人資訊是指可以識別或聯絡特定個人的資料。

您與國泰人壽或國泰人壽關係企業聯絡時,可能會要求您提供個人資訊。國泰人壽及其關係企業可能會在遵守本隱私權政策的前提下,相互共享本項個人資訊,也可能結合其他資訊,用於提供及優化我們的商品、服務及活動。您不一定要依照我們的要求提供個人資訊,但如果您選擇不提供,會導致我們在許多情況下無法為您提供商品或服務,或是無法回應您提出的問題。

以下舉例說明國泰人壽可能蒐集的個人資訊類型及其使用方式。

# (1) 資訊流說明

①手機 APP 向 APP SERVER 要求取得 QR Code。

②APP SERVER 使用 ID 向 SE Token API 索取 Token Key。

③SE Token API 查詢 ID 是否為會員,並向 SE Token info 取得 Token Key。

④SE Token API 回傳取得的 Token Key (或 失敗資訊)。

⑤APP SERVER 回傳取得的 Token Key。

⑥APP 依 Tokey Key 產生 QR Code。

⑦簽到員打開 SE Sign-in Mobile Web,進到 QR code scanner 功能,並掃瞄 ⑥ 的 QR Code。

⑧掃瞄後將 Token Key 傳入 SE QR Code check service。

⑨SE QR Code check service 向 SE Token info 確認 Token Key 是否正確(存在且未過期)。

⑩SE QR Code check service 取得客戶相關資料後將資料寫入報名表內。(客戶資料包含:姓名、手機,選碼顯示:如陳 O 安、0922xxx650)
