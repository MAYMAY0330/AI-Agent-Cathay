# 金控同仁兼任國壽職務於國壽辦公室進行資料蒐集分析疑義



法務室意見

緣　貴單位(客群經營部專案企劃科)因國泰金控(下稱金控)及本公司為業務需求，擬由金控同仁兼任客群經營部職務，兼任期間將使用本公司資料庫進行客戶個人資料之處理及利用，爰詢問該兼任同仁得否於金控辦公室辦公，或應於本公司辦公室辦公？本室謹覆意見如下。

按「金融控股公司或其子公司人員，除法令另有規定外，得兼為相關子公司之業務執行或商品銷售行為，並應先取得法定之特定資格或證照。」，金融控股公司及其子公司自律規範第17條第1項定有明文。

爰此，因本件所詢兼任人員之原任及兼任職務均為一般職員而非經理人，即與前揭規定無違，應無適法性問題(詳參本室99.4.2法律意見，附件1)，合先敘明。

查金控人員兼任本公司職務，其於執行本公司職務時，即為本公司之使用人，本公司授權其使用客戶資料庫，核屬公司內部對個人資料之處理及利用，應無適法性疑慮，餘請參本室106.2.17法律意見(附件2)。

另經詢 貴單位表示，本公司之客戶資料庫設有防火牆，僅本公司員工且有獲得授權者方可使用。爰此，於本公司有兼任職務之金控人員，如以本公司員工身分並於授權範圍內使用資料庫，應符合前揭個資法就「保有個人資料應採行適當安全措施」之規定。其如須於金控辦公室辦公，應尚無不可，惟本公司仍應注意個人資料之保密，並負管控之責，不得有洩漏或由非本公司業務相關人員取得之情。
