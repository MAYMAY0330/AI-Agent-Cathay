# 參與集團資料共享管理要點



國泰人壽參與集團資料共享管理要點

114 年 6月 6日訂定

權責單位：數據暨人工智慧發展部

(目的)

為規範本公司參與國泰金控暨子公司客戶資料共享機制（以下稱「資料共享」），保障客戶權益並提升本公司服務效能，爰訂定「國泰人壽參與集團資料共享管理要點」（以下稱「本要點」）。

(依據)

本公司應遵循《國泰金控暨子公司客戶資料共享政策》(以下稱「集團共享政策」)、《國泰人壽保險公司客戶資料共享政策》(以下稱「本公司共享政策」)及《國泰金融控股股份有限公司暨各子公司間資料共享管理辦法》(以下稱「管理辦法」)暨資料保護相關法令規定等參與資料共享。

(申請作業)

本公司應依管理辦法所訂程序向國泰金控數位數據暨科技發展中心數據暨人工智慧發展部(以下稱「金控管理單位」)申請相關作業。

本公司因參與資料共享取得之資料，其傳遞方式以及使用範圍應符合管理辦法所定之傳遞規範及使用範圍。

(權責單位)

本要點之權責單位為數據暨人工智慧發展部，並為本公司客戶資料共享相關業務之管理單位，除法令或內部規範另有規定者外，其職掌如下：

擔任本公司資料共享相關事務之聯繫窗口，配合集團共享政策、管理辦法、本公司共享政策及集團整體規劃執行資料共享之相關業務。

負責本公司申請客戶資料業務使用之審查，確保本公司相關資料匯入匯出之合理性，並維護資料之正確性及完整性，以及本公司共享資料取用之合理性評估，如涉及風險類資料之使用時，應依其性質會簽法令遵循單位、洗錢防制單位或風險管理單位。

訂定及修正本要點、管理本公司資料共享業務相關工作項目。

其他經核定應辦理事項。

(資料保密義務)

因參與資料共享所知悉他公司資料(包括但不限於機密性資料)之本公司員工(包括但不限於資料處理單位及業務單位同仁)，應負保密義務並簽訂《保密切結聲明書》，除法令規定、經主管機關或法院之要求或取得資料提供公司之同意者外，不得以任何方式洩漏予第三人，亦不得使用於申請目的外之用途，並應對前揭資料進行適當防止竊取、竄改、毀損及滅失之安全維護措施，以及遵守個人資料保護相關法令。

(客戶資料保護基礎)

本公司依《金融機構間資料共享指引》(以下稱「共享指引」)、集團共享政策、本公司共享政策及管理辦法蒐集處理利用客戶資料，應以《個人資料保護法》(以下稱「個資法」)為基礎於合理範圍內為之，並遵循下列規定：

辦理資料共享業務應有明確、妥適目的，並應符合相關法令規定及注意倫理道德，運用人工智慧評估客戶風險時，應注意避免產生潛在偏見之情形，運用人工智慧分析客戶資料後，應要求各公司不得對相同條件之客戶有不同差別待遇之情形。

共享之資料如屬身分核驗資料或負面資訊等，對客戶權益有較大影響性時，應利用交叉比對、評估與檢核或要求客戶說明等方式進行必要查證，或採取其他強化客戶資料保護措施(例如：提高調閱負面資訊之核定層級等)。

除法令另有規定外，資料共享得運用範圍應載明於客戶已簽署之契據文件及參與資料共享對象依法揭露於官網之個資應告知事項內容。

(客戶申訴及爭議處理)

本公司受理客戶因資料共享所提之申訴及爭議案件時，應注意以下事項：

受理客戶之申訴時應先確認客戶之身分，倘申訴者並非或無法確認為當事人本人或代理人者，或該申訴者拒絕提供身分確認文件者，應予拒絕以避免發生客戶資料洩漏予第三人之風險。

處理客戶之申訴，應遵循本公司《消費者保護方針》、《消費爭議處理程序》等客戶爭議處理相關作業規定，並主動通知客戶同意共享資料之公司。如當事人欲行使其個資法之相關權利，應依循本公司《個人資料管理辦法》相關規範辦理，並視必要性會辦相關單位評估處理。如當事人無法定或其他(包括但不限於另有契約約定)事由，得予以婉拒。

依規定應取得客戶同意方得使用者，若當事人表示拒絕集團資料共享，申訴受理窗口應立即通知客戶註記作業單位(智能客服中心)調整以停止該客戶資料於集團之共享使用。如涉及客戶資料共享之刪除作業，權責單位應於收到申訴受理窗口通知後，依循《國泰金融控股股份有限公司曁各子公司間資料共享管理辦法》相關規範辦理，並將結果回報金控管理單位。

申訴受理單位應依《金融消費者保護法》之規定於收受申訴之日起30日內為適當之處理，如涉及集團其他公司，應彙整各公司之處理結果後回覆提出申訴之客戶，但其他法令另有規定者，從其規定。

申訴或爭議案件涉及洗錢防制或打擊資恐之風險類資料時，應會辦洗錢防制最高主管，非經其同意不得揭露該等資料予申訴人或相關單位。

(內部控制)

本公司參與資料共享業務，應依據共享指引、個資法及其他經金融監督管理委員會等主管機關頒訂與金融機構間進行資料共享之其他法令函釋等規定，遵循包括但不限於執行本業務有關之系統日常維護、資料留存、權限設定、報表管理、共享結束後之處理等規範，且應依本公司各系統管理規範及個資管理相關規定辦理。

(未盡事宜)

本要點如有未盡事宜，悉依共享指引、個資法、國泰金控暨旗下子公司與本公司已簽訂之《客戶資料共享合作暨保密協議書》、相關法令、集團及本公司相關規定辦理。

(施行日期)

本要點呈總經理核准後施行，修正或廢止時亦同。
