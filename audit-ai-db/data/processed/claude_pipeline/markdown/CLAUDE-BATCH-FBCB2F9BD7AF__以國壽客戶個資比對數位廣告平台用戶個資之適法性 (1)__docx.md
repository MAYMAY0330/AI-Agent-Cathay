# 以國壽客戶個資比對數位廣告平台用戶個資之適法性 (1)



法務室意見

為提升準客經營業務曝光度、增加準客轉換率，並累積數位客群經營經驗，建立獲客模式，擬透過數位廣告平台(Facebook、Google聯播網、Appier等)向特定目標客群投放廣告。相關作業簡述如下：

將客戶個資（如手機號碼或電子郵件地址）經進行不可辨識且不可逆之雜湊作業後，傳送至數位廣告平台。

數位廣告平台之系統將不可辨識且不可逆之客戶個資，與雜湊後的數位廣告平台之用戶個資進行比對，並將符合的用戶新增至廣告受眾名單中。

數位廣告平台對廣告受眾名單投送公司委託刊登之廣告內容。

貴單位（即客群經營部）爰詢問上述作業流程是否符合個人資料保護法(下稱個資法)？本室謹復意見如下。

經遮蔽後之客戶個資，如與其他資料相互對照後即可識別特定之個人，仍屬個資法所稱之個人資料，而有個資法之適用：

按「一、個人資料：指自然人之姓名、出生年月日、國民身分證統一編號、護照號碼、特徵、指紋、婚姻、家庭、教育、職業、病歷、醫療、基因、性生活、健康檢查、犯罪前科、聯絡方式、財務情況、社會活動及其他得以直接或間接方式識別該個人之資料。」、「本法第二條第一款所稱得以間接方式識別，指保有該資料之公務或非公務機關僅以該資料不能直接識別，須與其他資料對照、組合、連結等，始能識別該特定之個人。」個資法第 2 條第 1 款及個資法施行細則第 3 條分別定有明文

職此，本件客戶個資雖經進行不可辨識且不可逆之雜湊作業而不能直接識別特定個人，惟如與其他資料相互對照、分析後，即可辨識該使用之特定個人者，依前揭規定，仍屬得以間接方式識別該特定人之個人資料，而有個資法之適用，合先敘明。（餘請參104年9月18日林政憲律師法律意見，附件1）

公司將客戶個資提供予數位廣告平台供行銷之用，應經客戶同意後方得為之：

按「非公務機關對個人資料之利用，除第六條第一項所規定資料外，應於蒐集之特定目的必要範圍內為之。…六、經當事人同意。」個資法第20條第1項定有明文。

爰此，倘公司蒐集客戶個人資料時，有告知客戶其個人資料將作為行銷之用，並徵得其同意(如附件2所示內容)，則前揭作業流程核屬就個人資料蒐集目的範圍內之利用，尚與個資法規定相符。

惟客戶受告知事項及同意範圍，仍需視具體個案而定，尚不可一概而論，建議　貴單位於使用客戶個資前再行確認，以維客戶權益。
