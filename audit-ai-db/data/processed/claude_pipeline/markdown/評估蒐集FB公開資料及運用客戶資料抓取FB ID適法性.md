## Page 2

# 法務室意見

一、按「非公務機關對個人資料之蒐集或處理,除第六條第一項所規定資料外,應有特定目的,並符合下列情形之一者:…三、當事人自行公開或其他已合法公開之個人資料。…」、「非公務機關對個人資料之利用,除第六條第一項所規定資料外,應於蒐集之特定目的必要範圍內為之。…」個人資料保護法(下稱個資法)第19條第1項第3款及第20條第1項定有明文。

二、次按「要保書之聲明事項係要保人或被保險人之授權及同意事項,其內容如下:…(三)本人(被保險人、要保人)同意(保險公司)就本人之個人資料,於「個人資料保護法」所規定之範圍內,有為蒐集、處理及利用之權利。…」人身保險要保書示範內容及注意事項第11點第3款定有明文。第按公司於保險單中之「個人資料保護法應告知事項」第一點「蒐集之目的」包括「人身保險」。

三、查 貴單位擬運用公司保戶既有聯絡資訊(手機號碼及Email帳號),至Facebook平台蒐尋、比對客戶之Facebook ID,並蒐集及分析客戶於該平台之公開資訊,以利提供差異化之行銷方案(應屬提供人身保險服務之目的範圍內),依前揭個資法規定,公司蒐集保戶自行公開於Facebook平台之個人資料,且目的係為日後提供差異化之客戶服務,應尚無適法性之疑慮。

四、惟該平台之公開資訊是否真實,提請 貴單位注意評估,以免影響差異化分析的結果。
